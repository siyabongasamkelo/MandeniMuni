# MandeniMuni

Mandeni Municipality Website

A modern, responsive municipal website designed to serve the community of Mandeni with easy access to essential services, announcements, and development updates. Built to highlight the resilience, unity, and growth of Mandeni, the platform provides quick links to municipal services, community news, load-shedding alerts, projects, and contact information.
This project focuses on accessibility, clean UI, and a user-first experience ‚Äî bringing government services closer to the people.
